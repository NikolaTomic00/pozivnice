# 🎉 Invitation Website – Modern RSVP Solution

## 📌 Problem

Planning events often comes with unnecessary stress. Traditional invitations create several common issues:

- Guests struggle to find event locations or navigate unclear addresses
- Tracking attendance becomes difficult due to lack of centralized RSVP
- Communication with guests is scattered and inefficient
- Organizers waste time confirming attendance manually

These challenges make event organization more complicated than it needs to be.

---

## 💡 Solution

This project presents a modern web-based solution for digital invitations.

Instead of relying on traditional methods, the platform provides:

- 📍 Clear event location details with easy navigation
- ✅ RSVP form for quick and simple attendance confirmation
- 📩 Centralized communication with all guests
- 📊 Better organization and tracking of responses

The goal is to simplify event planning and improve the overall experience for both hosts and guests.

---

## 🚀 Features

This is a modern, responsive website that includes all essential functionalities for digital invitations:

- 🎯 Clean and user-friendly UI/UX
- 📱 Fully responsive design
- 📝 RSVP form for guest confirmation
- 🗺️ Integrated location display
- 💬 Modern testimonial section (powered by Swiper.js)
- 💰 Pricing section (cenovnik)
- ✨ Smooth animations and interactive elements

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Swiper.js

To provide a simple, elegant, and efficient way to manage event invitations while eliminating common problems found in traditional approaches.
